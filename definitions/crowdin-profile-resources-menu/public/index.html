This template provides the foundation for a Crowdin app with a Profile Resources Menu module.

What's Included:

  • Express.js server with Crowdin SDK integration
  • Profile Resources Menu module configuration
  • JWT authentication handling
  • React JSON Schema forms support
  • Metadata storage for user preferences
  • Form data endpoints

Implementation Options:

Option 1: Custom HTML Interface

Replace this HTML file with your custom interface. Good for simple UIs or when you want full control.

Option 2: React JSON Schema Forms

Use declarative form schemas for automatic form generation. Uncomment the form configuration in index.js.

Next Steps:

1. Choose your implementation approach (HTML or JSON Schema)
2. Define your form schema or replace this HTML
3. Add your business logic in index.js
4. Test with your Crowdin user profile

Profile Resources Menu apps appear in the user profile menu and provide user-specific functionality.
