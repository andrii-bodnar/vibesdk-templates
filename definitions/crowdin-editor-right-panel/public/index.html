This template provides the foundation for a Crowdin app with Editor Right Panel module.

Editor Right Panel apps appear in the translation editor sidebar and provide context-aware functionality.

Example Actions

Implementation Guide

1. Replace this HTML with your custom panel UI
2. Use Crowdin Editor API to interact with translations
3. Most functionality is frontend-focused
4. Test within Crowdin translation editor

Available Editor API Methods:

// Insert text at cursor position
window.AP.editor.insertTranslation(text)

// Append text to current translation
window.AP.editor.appendTranslation(text)

// Replace entire translation
window.AP.editor.setTranslation(text)

// Get current translation
window.AP.editor.getTranslation()

// Get source string
window.AP.editor.getSourceString()
